# 🐾 ZooBuddy — Telegram-бот для владельцев домашних животных

Бот-помощник для ухода за питомцами: профили, напоминания, медкарта, дневник питания, советы и экстренная помощь.

## 📋 Функционал

| Раздел | Описание |
|--------|----------|
| 🐾 **Мои питомцы** | Профили питомцев (имя, вид, порода, дата рождения, вес, фото). Несколько питомцев на пользователя |
| ⏰ **Напоминания** | Кормление, прививки, ветеринар, груминг. Настраиваемые: разово / ежедневно / еженедельно / ежемесячно / ежегодно |
| 🏥 **Медкарта** | Прививки, визиты к ветеринару, учёт веса с графиком, хранение фото документов |
| 🍽 **Дневник питания** | Приёмы пищи, учёт воды, аллергии и непереносимости |
| 💡 **Советы** | Уход по видам (кошки, собаки, птицы, грызуны), FAQ, питание по возрасту |
| 🆘 **Экстренная помощь** | Поиск ветклиник по геолокации, памятки при отравлении, травме, перегреве |

## 🛠 Технологии

- **Python** 3.11+
- **aiogram** 3.x — асинхронный фреймворк для Telegram Bot API
- **SQLAlchemy** 2.0 — ORM с async-поддержкой
- **aiosqlite** — асинхронный SQLite-драйвер
- **APScheduler** — планировщик задач для напоминаний
- **matplotlib** — графики веса
- **FSM** (Finite State Machine) — управление диалогами

## 🚀 Установка и запуск

### 1. Клонируйте проект

```bash
git clone <repo-url>
cd ZOO_BOT
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/macOS:
source venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Скопируйте `.env.example` в `.env` и укажите токен бота:

```bash
copy .env.example .env
```

Отредактируйте `.env`:
```
BOT_TOKEN=ваш_токен_от_BotFather
DATABASE_URL=sqlite+aiosqlite:///./zoo_bot.db
```

### 5. Запустите бота

```bash
python bot.py
```

## 🔧 Для продакшена

Для продакшен-развёртывания рекомендуется:

1. Заменить SQLite на **PostgreSQL**:
   ```
   DATABASE_URL=postgresql+asyncpg://user:pass@localhost/zoo_bot
   ```
   И добавить в requirements: `asyncpg`

2. Использовать **Redis** для хранения FSM:
   ```python
   from aiogram.fsm.storage.redis import RedisStorage
   storage = RedisStorage.from_url("redis://localhost:6379")
   ```

3. Запускать через **systemd** или **Docker**
